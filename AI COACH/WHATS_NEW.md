## 🔧 Configuration

### Default: Built-in AI (Recommended)
The app uses the built-in AI coach by default. **No configuration needed!**

### Optional: Use Python Backend
If you want to use your own AI backend:

1. Edit `.env`:
```bash
VITE_USE_MOCK_BACKEND=false
VITE_API_URL=http://localhost:5000
```

2. Start Python backend:
```bash
python backend_simple.py
```

3. Restart frontend:
```bash
npm run dev
```
